# Kilroy

This project was my first experiment with creating Wallpaper Engine wallpapers. When applied through Wallpaper Engine, the text from the old "Kilroy was here" meme can be changed in the settings pane, as can the background colour and the main subject's colour. As the mouse is moved around the screen, Kilroy's eyes follow. This wallpaper works at any resolution, including when stretched across multiple monitors. This project was my introduction to Wallpaper Engine settings and use CSS animations.
